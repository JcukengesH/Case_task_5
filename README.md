# Case_task_5
Кейс-Задача №5 из учебной практики

Кейс-задача №5: Класс "Студент" (OOP C#)

Решение задачи по созданию класса "Студент" с использованием принципов ООП: инкапсуляция, свойства с валидацией, методы бизнес-логики.

## Функционал
- **Валидация данных**: Нельзя установить возраст < 16 или > 100, балл < 0 или > 10.
- **Информативность**: При ошибке программа сообщает, **в чьей именно** карточке произошел сбой.
- **Расчет статуса**: Автоматическое определение оценки ("Отлично", "Хорошо" и т.д.).
- **Дополнительно**: Возможность добавлять курсы.
